# foodshops-LPage
# FoodShops Landing Page  A modern, responsive marketing landing page for FoodShops – a food delivery marketplace connecting customers with local restaurants and vendors. 

##🌐 Live Preview
🔗 https://foodshops.vercel.app

## 📁 Project Structure
foodshops-theme/
├── index.html # Home page
├── about.html # About us
├── vendor.html # Vendor onboarding
├── delivery.html # Delivery partner onboarding
├── contact.html # Contact page
├── download.html # App download page
├── privacy.html # Privacy policy
├── terms.html # Terms & conditions
├── delete-account.html # Account deletion (Apple compliant)
├── css/
│ └── style.css # Complete styles
├── js/
│ └── main.js # All interactivity
└── images/ # Image assets

## 🎨 Features
- Fully responsive (mobile-first design)
- Smooth scroll animations
- Mobile hamburger menu
- FAQ accordion
- Form validation
- Intersection Observer scroll effects
- Lazy loading images

## 🛠️ Tech Stack
- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- Vanilla JavaScript (ES6)
- Font Awesome / Custom SVG icons
- Google Fonts (Poppins + Inter)

## 🔌 Backend Integration Points
- All forms currently use `onsubmit` with `alert()` placeholders
- Download buttons need App Store / Google Play URLs
- Newsletter signup needs email service integration
- Vendor/delivery applications need API endpoints

## 📦 Deployment
Deployed on Vercel. The site is static and requires no server-side processing.
